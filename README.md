# NDVI-Classification-Legend
|        NDVI | Class               |
| ----------: | ------------------- |
|      < 0.00 | Water / Built-up    |
| 0.00 – 0.20 | Barren Land         |
| 0.20 – 0.40 | Sparse Vegetation   |
| 0.40 – 0.60 | Moderate Vegetation |
|      > 0.60 | Dense Vegetation    |

# NDVI Interpretation Notes

NDVI values range from −1 to +1
Higher NDVI values indicate greater vegetation density and vigor
Thresholds may vary depending on season, sensor, and study area
Local calibration is recommended for crop-specific studies

# Usage

This classification legend is suitable for:
NDVI-based land cover classification
Vegetation health assessment
Change detection and temporal analysis
Agricultural and environmental monitoring
